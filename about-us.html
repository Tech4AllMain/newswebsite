Courses

Our club was founded on the basis that if someone requests a course, we will teach it for free! Here are  our current course offerings.

Video Editing

Video Editing is the manipulation and arrangement of video shots. It can be used to structure and present video information such as: movies, ads, and tv shows! In this course, you will learn how to use iMovie.

Java

In 1995, James Gosling, Mike Sheridan, and Patrick Naughton publicly released the Java language. Java is an object oriented high level programming language that has been used on laptops to supercomputers. In this course, you will learn Java and how to apply it.

Python

In 1991, Guido van Rossum publicly released python. Python is a high level programming language that emphasizes code readability. It is one of the fastest growing languages. In this course, you will learn how to code in Python and how to apply it.

Scratch

In 2003, MIT's media lab developed a block based visual programming language. It is great for beginners! In this course, you will learn how to use Scratch.

iOS App Development

In 2014, Apple released Swift. Swift is a programming language designed for iOS, macOS, iPadOs, watchOS, tvOS, and linux. It can be used to develop applications. In this course, you will learn how to program in swift, and make a couple of apps.
